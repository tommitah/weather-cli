# weather-cli

### Terminal app that outputs weather from openweathermap api

C#: to run - dotnet run

JS: to run - node index.js [city]
  - Bonus points for aliasing from the shell (eg. weather-cli = "node index.js")
  - City "flag" can be a spaced name (eg. New York)
  - No <-> needed, it's not actually a flag, just read straight from the arguments lol.
